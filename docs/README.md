# 다리 건너기

## 기능 목록

### 핵심
- [x] 다리 생성
  - 무작위 값(0, 1) 생성
- [x] 다리 이동
  - 이동 횟수 1 증가
  - 이동한 방향이 건널 수 있는 방향인지 판단
- [x] 게임 재시작 
  - 이동 횟수 0으로 초기화
  - 시도 횟수 1 증가
- [x] 이동 횟수가 다리 길이와 일치하는지 판단

### 입력
- [x] 다리 길이 입력
  - 3~20 숫자
- [x] 이동할 칸 입력
  - U, D
- [x] 게임 재시작/종료 여부 입력
  - R, Q

### 출력
- [ ] 시작 문구 출력
- [ ] 이동 결과 출력
- [ ] 최종 결과 출력
  - 이동한 다리 모습
  - 성공/실패 여부
  - 총 시도한 횟수

### 검증
- [ ] 다리 길이는 3~20 숫자
